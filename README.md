# my-snipaste 截图工具

> 轻量 Windows 截图工具，快捷键截图 + 文字/箭头/矩形标注。
> 专为公司内网设计，无需安装，下载即用。

---

## 快速开始

### 前提：安装 Python 3.8+

1. 下载：https://www.python.org/downloads/
2. 安装时**必须勾选 "Add Python to PATH"**

### 启动

```
git clone https://github.com/YU-JI-KUI/my-snipaste.git
cd my-snipaste
双击 start.bat
```

启动成功后右下角托盘出现图标，无需其他操作。

## 功能列表

### 截图

| 操作 | 说明 |
|------|------|
| `Ctrl+Alt+A` | 触发截图（全局快捷键，程序在后台也生效） |
| 拖拽鼠标 | 框选区域，选区外变暗、选区内高亮 |
| `ESC` / 右键 | 取消截图 |

### 编辑器

截图松开后自动进入编辑器，工具栏分两行：

**第一行（工具 + 操作）**

| 按钮 | 快捷键 | 说明 |
|------|--------|------|
| ✥ 移动 | 长按拖拽 | 拖动整个编辑器窗口 |
| T 文字 | 点击截图 | 在点击位置输入文字，Enter 确认，ESC 取消 |
| → 箭头 | 拖拽 | 绘制带箭头线段 |
| □ 矩形 | 拖拽 | 绘制空心矩形，框住重要区域 |
| ↩ 撤销 | `Ctrl+Z` | 撤销最后一个标注 |
| 💾 保存 | `Ctrl+S` | 弹出对话框保存为 PNG，默认到桌面 |
| ✓ 复制 | `Ctrl+C` | 复制截图到剪贴板 |
| ✕ 关闭 | `ESC` / 双击截图 | 关闭编辑器 |

**第二行（样式）**

- 8 种预设颜色（红/橙/黄/绿/蓝/紫/白/黑）
- 字号调节（8 ~ 72）

### 系统托盘

右键托盘图标：
- **截图**：等同于按 Ctrl+Alt+A
- **退出**：关闭程序

## 常见问题

**Q: 双击 start.bat 没有反应 / 窗口一闪而过**

运行 `python main.py`，查看终端输出，或查看同目录下的 `error.log`。

**Q: 快捷键没有反应**

部分安全软件会拦截键盘钩子，把 Python 加入信任列表，或改用托盘图标截图。

**Q: 复制到剪贴板失败**

需要 Windows 10 1903 以上版本，失败时请改用保存功能。

**Q: 重复双击 start.bat 启动了多个程序**

程序内置单实例检测，重复启动会自动退出，不会重复运行。

## 项目结构

```
main.py          # 程序入口
snip/
  app.py         # 主协调模块
  hotkey.py      # 全局热键
  overlay.py     # 截图遮罩与区域选择
  editor.py      # 截图编辑器
  tray.py        # 系统托盘
requirements.txt # 依赖
start.bat        # 一键启动
stop.bat         # 关闭程序
```
