# my-snipaste 截图工具

> 一个轻量的 Windows 截图工具，支持快捷键截图和添加文字标注。
> 专为公司内网环境设计，无需安装，下载即用。

---

## 功能特性

- **快捷键截图**：按 `Ctrl+Alt+A` 进入截图模式，拖拽鼠标选择区域
- **文字标注**：截图后点击任意位置输入文字，支持颜色和字号选择
- **保存图片**：`Ctrl+S` 保存为 PNG 文件，默认保存到桌面
- **复制到剪贴板**：`Ctrl+C` 一键复制截图，可直接粘贴到微信、钉钉等
- **系统托盘常驻**：程序运行在后台，右下角托盘图标随时截图
- **支持多显示器**：自动识别所有屏幕，跨屏截图无偏移

## 使用前提

只需要安装 **Python 3.8 或更高版本**。

### 安装 Python

1. 打开 https://www.python.org/downloads/
2. 下载最新版本（如 Python 3.12）
3. 运行安装包，**务必勾选 "Add Python to PATH"**

> 如果公司内网无法访问，请让 IT 同事帮忙安装 Python，或从其他渠道获取安装包。

---

## 快速开始

### 第一步：下载源码

在公司电脑打开命令行（Win+R，输入 cmd）：

```bat
git clone https://github.com/你的用户名/my-snipaste.git
cd my-snipaste
```

或者直接在 GitHub 页面点击 **Code → Download ZIP**，解压到任意目录。

### 第二步：启动程序

双击 **`start.bat`** 即可。

脚本会自动：
1. 检查 Python 是否安装
2. 安装所需依赖（`keyboard`、`Pillow`、`pystray`）
3. 在后台启动截图工具

启动成功后，右下角通知栏会出现程序图标。

## 使用方法

### 截图

| 操作 | 说明 |
|------|------|
| `Ctrl+Alt+A` | 进入截图模式 |
| 拖拽鼠标 | 选择截图区域，实时显示尺寸 |
| `ESC` 或 右键 | 取消截图 |

### 编辑截图

截图选区松开后，自动进入编辑窗口：

| 操作 | 说明 |
|------|------|
| 点击 **T 文字** 按钮 | 激活文字工具 |
| 点击截图区域 | 在该位置输入文字 |
| `Enter` 或 点击其他位置 | 确认文字 |
| 点击颜色色块 | 切换文字颜色 |
| 修改字号 | 调整文字大小 |
| `Ctrl+S` | 保存截图到桌面 |
| `Ctrl+C` | 复制截图到剪贴板 |
| `ESC` | 关闭编辑窗口 |

### 托盘菜单

右键点击右下角的程序图标：
- **截图**：手动触发截图（与快捷键效果相同）
- **退出**：关闭程序

---

## 关闭程序

双击 **`stop.bat`**，或在托盘图标右键选择 **退出**。

## 常见问题

**Q: 双击 start.bat 闪退了怎么办？**

A: 用命令行手动运行，查看报错信息：
```bat
cd my-snipaste
python main.py
```

**Q: 快捷键没有反应？**

A: 部分安全软件（如电脑管家）会拦截键盘钩子。可以：
- 将 Python 加入安全软件的信任列表
- 或通过托盘图标右键菜单截图

**Q: 截图有坐标偏移？**

A: 确保 Python 安装为 64 位版本，且系统 DPI 缩放为整数倍（如 100%、125%、150%）。

**Q: 中文文字显示乱码？**

A: 程序会自动使用系统内置的微软雅黑字体。如果仍有问题，请确保系统已安装中文字体。

**Q: 无法复制到剪贴板？**

A: 复制功能依赖 PowerShell，Windows 10 1903 以上版本支持。如果失败，请使用"保存"功能。
